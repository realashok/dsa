# Coding Exercises and Lab Work

## Coding Exercises

1. **Find Maximum of Two Numbers** – Write a function to determine the larger of two given integers.
2. **Check Prime Number** – Write a function that checks whether a number is prime.
3. **Reverse an Integer** – Write a function to reverse a given integer.
4. **Sum of Digits** – Write a function to compute the sum of digits of a number.
5. **Menu-driven Calculator** – Implement addition, subtraction, multiplication, and division using functions.
6. **Factorial (Recursive)** – Write a recursive function to calculate `factorial(n)`.
7. **Fibonacci (Recursive)** – Write a recursive function to calculate the nth Fibonacci number.
8. **Sum of Array (Recursive)** – Write a recursive function to sum all elements of an array.
9. **GCD (Recursive)** – Write a recursive function to find the GCD of two numbers.
10. **Reverse String (Recursive)** – Write a recursive function to print a string in reverse.
11. **Swap Using Pointers** – Write a program to swap two numbers using pointers.
12. **String Length Using Pointers** – Write a program to find the length of a string using pointers.
13. **Print Array Using Pointer Traversal** – Display array elements using pointer traversal.
14. **Sum of Integer Array (Pointer)** – Write a function that accepts a pointer to an integer array and returns its sum.
15. **Call by Reference (Pointers)** – Swap two integers using call by reference.
16. **Largest in Array (Pointers)** – Find the largest number in an array using pointers.
17. **Student Structure** – Define a `Student` structure (name, roll, marks) and input/display 3 students.
18. **Employee Structure** – Store and display employee info (ID, Name, Salary).
19. **Total & Average Marks** – Use an array of structures to compute total and average marks of students.
20. **Pass Structure to Function** – Pass a structure to a function and display its data.
21. **Nested Structures** – Create a structure containing another structure (e.g., Address inside Employee).
22. **Pointer to Structure** – Access members of a structure using the `->` operator.
23. **Array Traversal** – Write a program to traverse and print array elements.
24. **Insert Element in Array** – Insert an element at a given position.
25. **Delete Element in Array** – Delete an element from a given position.
26. **Linear & Binary Search** – Implement linear and binary search.
27. **Matrix Addition** – Add two 2D matrices.
28. **Matrix Transpose** – Find the transpose of a matrix.
29. **String Functions Using Pointers** – Implement custom `strlen()`, `strcpy()`, `strcmp()`.
30. **Character Frequency** – Count frequency of each character in a string.
31. **Reverse String (Recursion + Pointer)** – Reverse a string using recursion and pointers.
32. **Dynamic Array (malloc)** – Allocate array of N integers dynamically and find sum.
33. **calloc() Implementation** – Create an array of N integers initialized to 0 using `calloc()`.
34. **realloc() Example** – Expand an array from 5 to 10 elements.
35. **Dynamic Structure Allocation** – Allocate memory for a struct Student using `malloc()`.
36. **Memory Leak Example** – Demonstrate and fix a memory leak.
37. **Dynamic Linked List Node** – Create a simple linked list node using `malloc()`.
38. **Read Text File** – Read and display contents of a text file.
39. **Write to File** – Write student records to a file.
40. **Append to File** – Append content to an existing file.
41. **Binary File Storage** – Store and retrieve an array of structures in a binary file.
42. **Count Words, Lines, Characters** – Count basic statistics in a text file.
43. **Auto vs Static Variables** – Demonstrate difference inside a function.
44. **Extern Variable** – Use an extern variable from another file.
45. **Register Variable** – Demonstrate usage of register variables.
46. **Enum for Days** – Declare an enumeration for days of the week and print values.
47. **typedef for Structure** – Use typedef to simplify structure declarations.
48. **Enum-based Menu Program** – Write a menu-driven program using enums.
49. **Macro for Maximum** – Create a macro to find max of two numbers.
50. **Macro for Square** – Create a macro to compute square of a number.
51. **Command Line Sum** – Program to take two numbers from command line and print sum.
52. **Count Command-line Arguments** – Print number of arguments passed.
53. **Display File (CLI)** – Display contents of file provided as argument.
54. **Conditional Compilation** – Include debugging output using conditional compilation.
55. **Student List Management** – Manage list of students using structures and pointers.
56. **Recursive Structure Operations** – Perform recursive operations on structures.
57. **Record Management System** – Combine functions, pointers, and structures.

## Lab Work

### **Lab 1: Stack using Array**

Implementation of stack operations (push, pop, peek, display) using arrays.

### **Lab 2: Linear & Circular Queue using Array**

Implement enqueue, dequeue, and display in both linear and circular queue.

### **Lab 3: Recursive Algorithms**

* GCD
* Sum of Natural Numbers
* Tower of Hanoi

### **Lab 4: Linked List**

* Singly Linked List
* Doubly Linked List

### **Lab 5: Stack & Queue using Linked List**

Implement stack and queue operations using linked list nodes.

### **Lab 6: Tree Traversals**

* In-order
* Pre-order
* Post-order

### **Lab 7: Sorting Techniques**

* Insertion Sort
* Bubble Sort
* Quick Sort

### **Lab 8: Searching & Hashing**

* Sequential Search
* Binary Search
* Hash System

### **Lab 9: Graph Algorithms**

* Breadth-First Search (BFS)
* Kruskal’s Algorithm (MST)

### **Lab 10: Dijkstra’s Algorithm**

Implementation of Dijkstra's shortest path algorithm.
